# Nasa Images

## Getting Started

- run `yarn` to install all dependencies
- run `yarn dev` to run the project

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result, or check this [demo live link](https://nasa-images-shan.vercel.app/).

## Features

- View recent / popular images
- Image details view
- Free text search
- Search by location
- Search button will be disabled if no input is provided
- Search result pagination
